# Console Output Contract

## Required Section Order
1. Findings (High -> Medium -> Low)
2. Applied Fixes
3. Actual Quant Results
4. Open Questions / Assumptions
5. Optional Next Actions

## Findings Format
- [Severity] path:line
- Problem:
- Evidence:
- Impact:
- Action:

## Applied Fixes Format
- File changed:
- What changed:
- Why:
- Verification command:
- Verification result:

## Actual Quant Results Format
- Model/metric name:
- Value:
- Source log path:
- Before/after delta (if applicable):
